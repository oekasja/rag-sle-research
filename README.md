# Specialized Medical RAG-based LLM Dataset (SLE Case Study)

This repository contains the dataset and evaluation results for the research paper:  
**"The Impact of Knowledge Source Type on Specialized Medical RAG-based LLMs: A Case Study on Systemic Lupus Erythematosuss"**

## Repository Structure
- `list_of_med_docs`: Metadata of the 28 medical documents used (Textbooks, Journals, Guidelines).
- `question_bank.md`: The 45 medical queries used for testing.
- `scoring_results.csv`: Anonymized raw scores from 3 medical residents.
- `system_config.json`: Technical parameters of the RAG pipeline.

## Experts Profile
The evaluation was conducted by three **Internal Medicine Residents** from the **Internal Medicine Residency Program** at **Brawijaya University, Indonesia**.
